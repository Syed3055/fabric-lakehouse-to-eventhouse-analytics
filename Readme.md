# 🚀 Fabric Lakehouse to Eventhouse Analytics

## 📌 Overview
This project demonstrates an end-to-end data analytics workflow using **Microsoft Fabric**. Data from CSV files is ingested into the **Lakehouse**, processed, and then analyzed in **Eventhouse using KQL**.

---

## 🏗️ Workflow

CSV Files  
→ Lakehouse Tables  
→ Eventhouse (KQL Analysis)  
→ Insights  

---

## 🔹 Steps Performed

1. Uploaded CSV files into Microsoft Fabric  
2. Created tables in Lakehouse  
3. Accessed data in Eventhouse  
4. Performed analysis using KQL queries  
5. Generated insights from structured data  

---

## 🛠️ Technologies Used

- Microsoft Fabric  
- Lakehouse  
- Eventhouse  
- KQL (Kusto Query Language)  

---

## 🎯 Objective
To analyze structured data by integrating **Lakehouse and Eventhouse** and performing efficient querying using **KQL**.
